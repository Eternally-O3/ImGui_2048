# ImGui_2048
 2048 游戏 - 基于 C++ 的 DirectX9 和 ImGui 实现

	前置条件
	编译和运行该项目，需要以下环境配置：

	C++17或更高
	Win32，多字节字符集
	DirectX9 SDK
	ImGui
	Windows 操作系统
